### 🚀 Featured Projects

#### 🌐 Web Applications & Platforms
*   **[CompressBit](https://compressbit.com)** — *Privacy-First Client-Side Compressor*
    > A professional web utility for compressing PDF and image files. All processing happens entirely client-side in the browser's memory using JavaScript and WebAssembly, ensuring files are never uploaded to a server.
    > `JavaScript` `WebAssembly` `HTML5` `Wasm`
*   **ViBe** — *Interactive Virtual Classroom Hub*
    > An online learning and student engagement platform featuring quick quizzes, smart presence checks, and real-time engagement monitoring tools to foster active virtual classrooms.
    > `React` `Node.js` `Express` `MongoDB`
*   **ClassConnect** — *Virtual Classroom Manager*
    > A lightweight, PHP-based classroom management system designed for seamless teacher-student interactions, grading, and announcement sharing.
    > `PHP` `MySQL` `Bootstrap`

#### 🔐 Biometrics & Blockchain (Web3)
*   **Biochain Vote & Scanner** — *Biometric-Secured Blockchain Voting*
    > A secure voting portal utilizing blockchain technology and hardware fingerprint verification (integrating Nitgen/NBioBSP SDKs) to guarantee zero-trust identity verification.
    > `Solidity` `Web3.js` `MERN Stack` `Biometric SDK`
*   **Fingerprint Service** — *Universal Biometric Authentication API*
    > A localized backend service interfacing directly with hardware fingerprint SDKs, exposing clean APIs to handle user registration and verification for modern web clients.
    > `Node.js` `Express` `SDK Integration`

#### 🤖 Artificial Intelligence & Machine Learning
*   **PDF & PPT Summarizer for Exams** — *NLP Academic Summarizer*
    > An NLP-powered tool designed to digest exam materials (lecture notes, textbooks, slides) and instantly generate high-quality summaries, flashcards, and practice questions.
    > `Python` `NLP` `Machine Learning`

#### 🔌 IoT & APIs
*   **NAMSTE-ICD-API (AYUSH-Connect)** — *FHIR-Compliant Healthcare Bridge*
    > A health-tech RESTful API built for the Smart India Hackathon (SIH), designed to bridge Traditional Indian Medicine (AYUSH) database records with modern Electronic Health Records (EHR).
    > `FHIR` `Node.js` `RESTful API` `Healthcare Tech`
*   **IoT Water Irrigation System** — *Automated Crop Irrigation Controller*
    > An automated smart crop-watering controller built on hardware microchips, using scheduled programs and soil moisture levels to optimize water usage.
    > `C++` `Arduino` `Hardware Programming`
